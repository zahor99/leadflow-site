Openclaw

In short

Gergely proposes a full-stack AI application for Openclaw combining a pgvector RAG backend with source-cited Q&A, LangGraph agent orchestration for article/ebook research, and a containerized Kokoro TTS layer exposed via REST API. Phase 1 delivers a working RAG + Q&A core the client can run immediately. Rate is $95/hr.

Architecture I'd propose

  • Agent Layer — LangGraph / Claude tools
  • RAG Backend — pgvector + reranker
  • FastAPI — retrieval & agent API
  • Next.js Editor — article / ebook assembly
  • Kokoro TTS — Docker + REST, all locales
Assumes: Assumes source documents are web URLs or uploadable files — not a proprietary CMS · Assumes Kokoro container can run on your infra or a cloud VM · Assumes article/ebook output is text-based export, not a custom binary format — tell me where I'm wrong and the design adapts.
